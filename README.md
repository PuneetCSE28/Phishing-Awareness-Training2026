# CodeAlpha_PhishingAwarenessTraining

## 📌 Overview
This project is **Task 2: Phishing Awareness Training**, completed as part of the **CodeAlpha Cyber Security Internship**. It delivers a complete, presentation-ready security awareness training module that teaches non-technical users how to recognize, avoid, and report phishing attacks.

## 🎯 Objective
- Explain what phishing is and why it's the leading cause of security breaches
- Break down the psychological tactics attackers use (urgency, fear, authority, curiosity)
- Cover the major phishing types: email, spear phishing, whaling, smishing, vishing, clone phishing
- Show annotated, red-flagged examples of a fake email and a fake login page
- Walk through two real-world phishing case studies and their lessons
- Provide a clear Do's/Don'ts checklist and an incident-response checklist
- Test retention with a short interactive quiz (with answers)

## 🛠 Tools & Technologies
- **Content design:** PowerPoint-compatible slide deck (`.pptx`)
- **Generation:** Node.js + `pptxgenjs`
- **QA:** LibreOffice (render/convert), `markitdown` (content check), custom OOXML validator

## ✅ How to Use
1. Open `Phishing_Awareness_Training.pptx` in PowerPoint, Google Slides, or LibreOffice Impress.
2. Present as-is, or adapt for a live/interactive training session (e.g. read the quiz aloud and ask for a show of hands).
3. Optionally record a walkthrough video for submission.

## 📚 Key Takeaway
Phishing succeeds by exploiting **trust and urgency**, not technical flaws. Slowing down, verifying requests through a separate known channel, and reporting anything suspicious stops the vast majority of attacks.
